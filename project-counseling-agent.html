PROJECT 01 · COUNSELING AGENT

心理咨询多 Agent 系统

这是我在大模型算法实习期间深度参与的核心项目，重点围绕心理咨询场景中的数据工程、模型微调、Agent 协作、知识增强与前端交互展开。目标不是单一问答能力，而是构建一个能在多轮咨询中维持上下文连续性、专业性与可解释性的智能系统。

Qwen3-32B / Qwen2.5-72B Dify / LangChain Prompt Engineering Long-term / Short-term Memory Vue.js

我负责的部分

  • 处理 40 万条心理咨询对话数据，包括清洗、脱敏、增强、标准格式转换与质量检测。
  • 使用 LlamaFactory 完成 Qwen 系列模型的 SFT 微调，并参与评测、部署与效果迭代。
  • 设计 Router、Information、Diagnosis、Recommendation 多 Agent 协作结构。
  • 构建心理学知识库与长期/短期记忆机制，提升多轮咨询的连贯性与可靠性。
  • 使用 Vue.js 实现用户交互、历史记录与 Agent 分析可视化界面。

我最关心的系统问题

  • 咨询场景如何在专业判断、共情表达与安全边界之间取得平衡。
  • 多 Agent 之间如何分工，而不是简单叠加多个调用步骤。
  • 记忆机制如何既保留上下文，又避免错误信息在多轮对话中被放大。

结果与收获

这个项目让我真正把大模型训练、知识库、Prompt、Agent 工作流与前端产品体验连成一条完整链路，也更明确了我对“可部署、可迭代智能系统”的长期兴趣。
