PROJECT 02 · MAAR-AGENT

多源增强多模态 RAG 系统

MAAR-Agent 面向复杂问答中的三类核心问题而设计：检索不足、多模态利用率低、幻觉率高。系统尝试把感知、问题分析、多源检索、重排序、推理生成、事实验证与自我纠错组织成一条更完整的 Agent RAG 链路。

Llama-3.2-11B-Vision-Instruct CLIP BGE-large / BGE-reranker FAISS vLLM

系统主线

  • 感知：识别问题涉及的文本、网页与图像线索。
  • 问题分析：判断需要哪些检索源与后续推理策略。
  • 多源检索：融合文本检索、网页检索与图像检索。
  • 重排序与生成：通过 reranker 聚焦证据，再进入推理回答。
  • 事实验证与自我纠错：用一致性验证降低幻觉率。

关键亮点

  • 不仅关注“能否答出来”，更关注“证据是否可信”。
  • 把 CLIP 一致性、语义一致性和事实一致性串进验证环节。
  • 从单轮检索问答延展到更像智能体工作流的多阶段系统。
